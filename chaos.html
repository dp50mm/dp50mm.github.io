Chaos theory workshop

This page is 'dynamic': meaning that I will be putting up more information over the following two weeks leading up to the workshop.

This workshop will be about playing with awesome algorithms. I will prepare a couple of interactive web pages for the various algorithms so that you can immediately start playing with them without much coding experience. For the more experienced programmers all of the code will be on github so that you can start hacking away. Actually I hope and expect you will be creative!

Why a workshop on chaos theory?

To develop some intuition in -

  • Non-linearity
  • Emergence
  • Chaos

- through play.
